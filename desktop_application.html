Project Description

This project involved developing a graphical interface for a nutrition and exercise platform that provides personalized workout and meal plans based on user input. The design focused on a modular layout, enabling users to easily access, navigate, and modify specific components such as workout routines and dietary preferences. This approach enhanced usability, flexibility, and scalability, allowing the platform to adapt seamlessly to different user needs and future feature expansions.
